# [Project Euler](https://projecteuler.net/) in C++

This project is a mere test vehicle for `cmake`, `ctest` and `catch2`.

## Installation

```shell
git clone https://github.com/pbing/project-euler-cpp.git
cd project-euler-cpp
cmake -G Ninja -B build
cd build
cmake --build .
```

## Run tests

```shell
ctest
```
or
```shell
./tests/tests
```

## Solve all problems

```shell
./src/main
```
